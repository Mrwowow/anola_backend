üöÄ Quick Start

Base URL

https://anola-backend.vercel.app

Authentication

Most endpoints require JWT authentication. Include the token in the Authorization header:

Authorization: Bearer YOUR_JWT_TOKEN

Example: Login

curl -X POST https://anola-backend.vercel.app/api/auth/login \ -H "Content-Type: application/json" \ -d '{"email": "admin@anolalinks.com", "password": "Possible@2025"}'

üîë Key Endpoints

  • POST /api/auth/login - User login
  • POST /api/auth/register - User registration
  • GET /api/super-admin/dashboard - Admin dashboard (requires auth)
  • GET /api/super-admin/users - List all users (requires auth)
  • GET /api/patients - List patients (requires auth)
  • GET /api/providers - List providers (requires auth)
  • GET /api/appointments - List appointments (requires auth)
  • GET /api/transactions - List transactions (requires auth)
  • GET /health - API health check
